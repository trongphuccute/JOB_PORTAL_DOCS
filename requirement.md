# System Requirements - JobPortal

---

## 1. User Roles

- Guest
- Job Seeker
- Employer
- Admin

---

## 2. Functional Requirements

### Authentication
- Register
- Login / Logout

---

### Job Seeker

- Create profile
- Upload resume
- Search jobs
- View job detail
- Apply job
- Save job

---

### Employer

- Create company
- Post job
- Edit job
- Delete job
- View applicants

---

### Admin

- Manage users
- Manage jobs

---

## 3. Non-Functional Requirements

### Performance
- Fast response (<3s)

### Security
- CSRF protection
- Password hashing

### Usability
- Responsive UI (Bootstrap)

---

## 4. Database

Tables:
- users
- profiles
- companies
- jobs
- applications
- resumes
- saved_jobs

---
