# Development Workflow - JobPortal

---

## 1. Git Workflow

Branches:

- main → production
- develop → development
- feature/* → features

Example:
- feature/login
- feature/job-management

---

## 2. Development Process

1. Create branch
2. Code feature
3. Test locally
4. Commit
5. Push GitHub
6. Merge develop

---

## 3. Coding Workflow

- Define models
- Run migrations
- Create views
- Create templates
- Add JS
- Test

---

## 4. Feature Example (Apply Job)

1. Click Apply
2. JS send POST
3. Backend validate
4. Save DB
5. Return response

---

## 5. Testing

- Manual test
- Validate form
- Check database
